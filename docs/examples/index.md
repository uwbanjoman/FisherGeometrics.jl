# Examples

Runnable Julia scripts demonstrating the FisherGeometrics framework.
All scripts run in seconds on a laptop.

```bash
julia --project examples/spacetime_simulation.jl
julia --project examples/yang_mills.jl
julia --project examples/black_holes.jl
julia --project examples/gravitational_waves.jl
julia --project examples/cosmology.jl
julia --project examples/derive_K.jl
```
